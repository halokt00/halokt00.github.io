Khu Vực Tu Luyện

Chức năng game sẽ được xây dựng ở bước tiếp theo.
Hiện tại bạn có thể thử chức năng Lưu Data (tải về máy) và Tải Save (tải lên từ máy) để kiểm tra hệ thống.

Nhật Ký Cập Nhật

  • [Mới] Hệ thống tải save từ file `.json` trên máy tính.
  • [Mới] Nút "Lưu Data" cho phép tải file save về máy để lưu trữ an toàn.
